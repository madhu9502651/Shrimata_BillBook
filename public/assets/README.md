# Assets Directory

## Required Files

### logo.png
Place your company logo file here as `logo.png`

**Requirements:**
- File name: `logo.png`
- Recommended size: 400x400 pixels or larger (square format works best)
- Format: PNG with transparent background recommended
- Used in: Receipt header and watermark

The logo will:
- Appear in the top-left of receipts (80x80px display size)
- Serve as a background watermark in receipts (large, very low opacity ~6%)
